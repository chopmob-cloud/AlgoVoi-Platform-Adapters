# Xero — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against Xero invoices, automatically generating AlgoVoi hosted checkout links when invoices reach AUTHORISED status.

Full integration guide: [xero.md](../xero.md)
