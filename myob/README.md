# MYOB — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against MYOB AccountRight and Essentials invoices, automatically generating AlgoVoi hosted checkout links for unpaid invoices.

Full integration guide: [myob.md](../myob.md)
