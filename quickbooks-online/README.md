# QuickBooks Online — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against QuickBooks Online invoices, automatically generating AlgoVoi hosted checkout links when invoices are created.

Full integration guide: [quickbooks-online.md](../quickbooks-online.md)
