# FreshBooks — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against FreshBooks invoices, automatically generating AlgoVoi hosted checkout links when invoices are created or sent.

Full integration guide: [freshbooks.md](../freshbooks.md)
