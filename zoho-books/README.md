# Zoho Books — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against Zoho Books invoices, automatically generating AlgoVoi hosted checkout links when invoices are created.

Full integration guide: [zoho-books.md](../zoho-books.md)
