# Sage Business Cloud — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against Sage Business Cloud invoices via scheduled polling, as Sage does not support push webhooks.

Full integration guide: [sage-business-cloud.md](../sage-business-cloud.md)
