# Wave Accounting — AlgoVoi Payment Adapter

Accepts USDC on Algorand and aUSDC on VOI as payment against Wave invoices via the WaveApps GraphQL API, automatically generating AlgoVoi hosted checkout links for unpaid invoices.

Full integration guide: [wave.md](../wave.md)
